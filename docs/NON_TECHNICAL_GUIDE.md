## 1. What is EpiPredict Kenya AI?

EpiPredict is an early-warning system for diseases. Think of it like a weather forecast, but for outbreaks. Instead of telling you it *is* raining, it tells you it *is going to* rain in two weeks, giving you time to buy umbrellas.

**Goal:** To move Kenya's healthcare system from **reacting** to crises to **preventing** them.

## 3. Security & Privacy (For Compliance Officers)

We know that health data is sensitive. We have built EpiPredict with "Privacy First" principles:

- **What we store**: We only store aggregated trend data (e.g., "50 cases of flu in Nairobi") rather than individual patient names.
- **How it's locked**: We use **Encryption** (like bank-grade security) to scramble data so hackers cannot read it.
- **Who can see it**: Only authorized personnel with a login can access the dashboard.
- **Compliance**: Our system is designed to adhere to Kenya's **Data Protection Act (2019)** regarding health data processing.
